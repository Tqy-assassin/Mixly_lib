>Author :CXSNRobot
>
>website: 
>
>Test on mixly V0.998

###使用方法

* 1.打开mixly软件，点击"导入库/Import"。
* 2.选中CXSNx10文件夹中的CXSNx10.xml文件，提示导入成功即可。
